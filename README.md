## Tech Stack

**Server:** Nestjs, TypeORM, Sockets.io, Swagger

- Despliegue: [Backend - Swagger](https://contatec.herokuapp.com/)

## Instalacion

```bash
$ npm install

# Agregar variables de entorno (similar a .env.example)
$ touch .env

# (Crear base de datos como se especifique en .env)
```

## Ejecutando la App

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
